/////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
Comandos para iniciar proyecto en Django
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


Crear el proyecto Django ---> django-admin startproject name-project
Crear app dentro del proyecto ---> python manage.py startapp name-app
General la migracion de la aplicacion ---> python manage.py migrate



/////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
Comandos basicos Django
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////

Correr el servidor ---> python manage.py runserver
Crear migracios de modelo a SQL ---> python manage.py makemigrations
General la migracion de la aplicacion ---> python manage.py migrate
